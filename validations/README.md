# validations

This folder contains validation scripts and generated validation outputs.

## What it contains
- Validation utilities/scripts (for example `maps.py`, `run_maps_mrt_example.py`, dataset-specific checks).
- Dataset-level subfolders with generated figures (timeseries/maps) grouped by dataset.

## Role in the workflow
- Stores validation products used to compare datasets, inspect consistency, and support dashboard reporting.
