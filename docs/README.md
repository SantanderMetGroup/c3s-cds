# docs

This folder contains the GitHub Pages site used to browse catalogues and dashboard summaries.

## What it contains
- Static pages (`index.html`, `dataset.html`, `catalogue-viewer.html`).
- Frontend assets (`script.js`, `style.css`).
- `data/`: exported catalogue CSV files used by the dashboard.
- `img/`: rendered assets (heatmaps and validation figures).

## Role in the workflow
- Publishes a navigable view of catalogue status and dataset coverage.
- Mirrors outputs generated from `catalogues/` and `validations/` for web visualization.
